# Back End Node.js

- Installation des dépendances NPM dans le contexte technique du container du service "api", répliquées dans le volume local

`docker compose run --rm api npm install`

- Lancement du service

`docker compose up`

ou

`docker compose up -d`

- test de l'API

`curl localhost:3333`
